# Key Object Icons — Iconos de objetos clave

## Uso en la app
Cada estación otorga un objeto o fragmento. Estos iconos viven en el inventario, el mapa y el final.

## Partes que necesita este layout
- forma base
- borde por itinerario
- símbolo central
- versión collected
- versión missing

## Patrón visual
- medalla circular o rectangular según objeto
- contorno claro
- icono central muy legible
- glow sutil en collected

## Oceanogràfic
- piezas del mapa
- animales marinos
- gotas, corales, brújulas y fragmentos

## Museu
- diapositivas
- engranajes
- símbolos de ciencia
- cristal, teatro y experimentación

## Reglas
- tamaño pequeño, alta legibilidad
- bordes limpios
- el símbolo debe entenderse sin texto
