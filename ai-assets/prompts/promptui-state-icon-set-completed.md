# UI: State Icon — Completed

## Style Context

**Visual direction:** Oceanogràfic Jumanji — marine expedition marker. Glass, shell, pearl, sonar language. Readable at 24px. Dark and light surface compatible.

**Palette:**
- Jade: `#56D0B1` | Deep teal: `#0E8277`
- Ancient gold: `#F1C96B` | Ice: `#BFEFFF`

**Rules:** No text. Transparent background. Consistent set of 4 states. One icon only.

## Grok Prompt

**Style:** Flat vector icon, minimalist UI glyph, marine expedition marker, reward language.
**Subject:** A single "completed" state icon for the route step progress. A polished jade crystal or gem shape with a subtle checkmark or star symbol inside. The icon is round with a rich jade and deep teal gradient. Gold sparkle dots around the edge suggest achievement.
**Palette:** Base jade `#56D0B1` blending to deep teal `#0E8277`, gold `#F1C96B` sparkle accents at 60% opacity, ice `#BFEFFF` highlight.
**Mood:** Completed, achieved, rewarding. Satisfying to see.
**Composition:** 96x96 icon, centered. Reads clearly at 24px. Transparent background.
**Technical:** Clean vector shapes, gem-like gradient, subtle sparkle accents. Part of a 4-icon set (locked, active, completed, done).

## Negative Prompt

no text, no readable letters, no watermark, no logo, no clutter, no photorealistic, no flat generic UI circle, no overly flashy effects

## Output

- Master: 96x96
- Additional: 48x48, 32x32
- Format: PNG with transparent background
