# Print: Passport Crystal Sheet

## Style Context

**Visual direction:** Oceanogràfic Jumanji — printable crystal sheet matching station rewards. Clean grid with even spacing for labels or stickers. Marine expedition sticker board style.

**Palette:**
- Deep background: `#04131D` | Elevated surface: `#1B5A6D`
- Ancient gold: `#F1C96B` | Jade: `#56D0B1`

**Rules:** No text. Clean grid layout. A4 portrait print quality.

## Grok Prompt

**Style:** Print layout design, sticker sheet, clean grid, vector illustration.
**Subject:** A printable crystal sheet that matches the station rewards used in the UI. A clean grid of 18 circular slots arranged in 6 columns x 3 rows. Each slot is a recessed glassy circle with subtle ocean marks (tiny sonar rings, wave lines) around it. The background is dark textured parchment. Enough spacing between slots for cutting or labels.
**Palette:** Dark background `#04131D`, elevated surface `#1B5A6D` for the slots, gold `#F1C96B` micro-accents on the ocean marks, jade `#56D0B1` for subtle decorative dots.
**Mood:** Clean, organized, expedition sticker board.
**Composition:** A4 portrait 2480x3508. Even grid, consistent spacing, quiet margins.
**Technical:** Print quality 300 DPI. No baked text. Clean slots for crystal placement.

## Negative Prompt

no text, no readable letters, no watermark, no logo, no clutter, no photorealistic, no generic school worksheet, no photo collage

## Output

- Size: 2480x3508 px (A4 portrait, 300 DPI)
- Format: PNG
- Background: dark textured sheet
