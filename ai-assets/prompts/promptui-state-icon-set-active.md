# UI: State Icon — Active

## Style Context

**Visual direction:** Oceanogràfic Jumanji — marine expedition marker. Glass, shell, pearl, sonar language. Readable at 24px. Dark and light surface compatible.

**Palette:**
- Sea surface: `#123B4E` | Ancient gold: `#F1C96B`
- Jade: `#56D0B1` | Deep teal: `#0E8277`
- Ice: `#BFEFFF`

**Rules:** No text. Transparent background. Consistent set of 4 states. One icon only.

## Grok Prompt

**Style:** Flat vector icon, minimalist UI glyph, marine expedition marker, glowing sonar language.
**Subject:** A single "active" state icon for the route step progress. A glowing glass pearl or sonar ping with a pulsing ring around it. The icon is round with a warm teal-to-jade gradient. A bright inner glow suggests current activity. A thin gold ring encircles it.
**Palette:** Base `#123B4E`, jade `#56D0B1` glow, gold `#F1C96B` ring at 70% opacity, ice `#BFEFFF` center highlight.
**Mood:** Active, in-progress, current. Bright enough to draw attention.
**Composition:** 96x96 icon, centered. Reads clearly at 24px. Transparent background.
**Technical:** Clean vector shapes, inner glow effect, pulsing ring. Part of a 4-icon set (locked, active, completed, done).

## Negative Prompt

no text, no readable letters, no watermark, no logo, no clutter, no photorealistic, no flat generic UI circle, no overly bright colors, no animation (static image)

## Output

- Master: 96x96
- Additional: 48x48, 32x32
- Format: PNG with transparent background
