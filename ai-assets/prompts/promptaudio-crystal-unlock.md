# Audio: Crystal Unlock

## Style Context

**Visual direction:** Oceanogràfic Jumanji — aquatic magical reward sound. Marine expedition audio feedback.

**Design principles:** Simple, clean, family-friendly. Must work as UI feedback on mobile web.

## Grok Prompt

**Style:** Short clean sound design, aquatic magical SFX, foley.
**Subject:** A reward sound for collecting a crystal in the Oceanogràfic Jumanji UI. A soft sonar ping opens the sound, followed by a glass shimmer with harmonic sparkle, a bubbling rise, and a warm treasure-like finish with a gentle resonant tail.
**Palette (audio):** Soft attack, medium sustain, clean decay. Frequency range: 400Hz–8kHz. Warmth in the low-mids, sparkle in the highs.
**Mood:** Magical, rewarding, aquatic. Satisfying but not jarring. Family-friendly.
**Technical:** No voice, no melody, no long silence, no clipping, no distortion. Must loop cleanly if needed. Stereo, 44.1kHz, 16-bit.
**Length:** 0.8 to 1.2 seconds.

## Negative Prompt

no voice, no vocals, no melody, no music, no long silence, no clipping, no distortion, no noise, no harsh frequencies

## Output

- Length: 0.8–1.2 seconds
- Format: WAV (master), MP3 + OGG (exports)
- Channels: Stereo
- Sample rate: 44.1kHz, 16-bit
