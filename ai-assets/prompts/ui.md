# UI

## Global negative prompt

```text
no text, no watermark, no logo, no unreadable tiny detail, no flat generic UI, no neon sci-fi, no clutter, no photorealistic people
```

## ui/mission-panel-surface

Create a large mission panel surface for the main route screen. The asset should look like a carved marine control board or glass-reinforced expedition panel: dark teal frame, subtle gold edging, wet reflections, readable interior zone for station copy and challenge controls, and slightly heavier corners to anchor the layout. The asset must support both desktop and mobile crops. Placement: station card and any large mission panel surface. Output: 1600x1200 master with transparent or masked edges, plus webp/png exports sized for desktop and mobile crops.

## ui/final-panel-surface

Create a final screen panel surface for the completion view. The surface must feel more luminous and ceremonial than the mission panel: deeper glow, more gold, more ocean light, and a clear area for the honor badge, final message and action buttons. Keep the edges decorated but never crowded. Placement: final screen card and printable certificate base. Output: 1600x1200 master, webp/png exports.

## ui/guardian-banner

Create a wide guardian banner for the riddle intro and treasure banner. The banner should be 1440x360 with a compact 720x240 variant. Style it like a marine expedition ribbon carved from glass and coral: strong readable middle zone for copy, decorative sonar pulses and shell fragments at the ends, no text baked into the image. Placement: guardian intro, treasure banner, status ribbon. Output: 1440x360 and 720x240, webp/png.

## ui/state-icon-set

Create a reusable set of small state icons for route steps and riddle steps. Produce four consistent icons: locked, active, completed and done. They must remain readable at 24 px, work on dark and light surfaces, and feel like marine expedition markers rather than generic UI dots. Use glass, shell, pearl or sonar language rather than flat circles. Placement: route list, riddle blocks and progress cues. Output: 96x96 transparent PNG masters for each state, plus 48 and 32 exports.

## ui/adult-help-badge

Create a small icon badge for the adult help button. It should feel like a helpful expedition marker, not a warning: a life ring, lantern, beacon or guardian cue in marine glass style. The icon must fit inside a button with text and remain legible at 24 px. Placement: adult help button and optional tooltip marker. Output: 256x256 transparent PNG master with 64 and 32 exports.
