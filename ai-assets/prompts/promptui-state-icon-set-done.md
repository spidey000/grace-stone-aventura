## Grok Prompt

**Style:** Flat vector icon, minimalist UI glyph, marine expedition marker, finality language.
**Subject:** A single "done" state icon for the route step progress. A filled gold medal or sealed scroll shape. The icon is round with a deep teal and muted navy gradient. A gold star or crest badge fills the center, indicating final completion. Less bright than "completed" — settled and definitive.
**Palette:** Base `#1B5A6D` muted teal, gold `#F1C96B` and amber `#D89A2B` for the badge symbol, deep teal `#0E8277` for depth.
**Mood:** Finished, archived, done. A final settled state, not drawing attention.
**Composition:** 96x96 icon, centered. Reads clearly at 24px. Transparent background.
**Technical:** Clean vector shapes, solid symbol, no glow effects. Part of a 4-icon set (locked, active, completed, done).
