# Brand

## Global negative prompt

```text
no readable text, no watermark, no logo, no random letters, no clutter, no photorealistic poster, no generic app icon
```

## brand/itinerary-oceanografic-badge

Create a square itinerary badge for the Oceanogràfic card in the lobby selector. The badge must feel like a premium expedition token made for a marine adventure: glass ring, wave crest, compass-like eye of the ocean, subtle coral or turtle silhouette, teal and gold accents, and a strong readable center. It should work as a card icon, a small badge and a print mark. Placement: itinerary selector, final certificate, passport cover. Output: 1024x1024 master, transparent PNG or dark plaque variant, with smaller 512 and 256 exports.

## brand/itinerary-museu-badge-secondary

Create a secondary square itinerary badge for the Museu route that still belongs to the same Oceanogràfic Jumanji visual system. Make it feel scientific and exploratory, but marine-compatible: glass vial, sonar line, notebook stamp or lab marker mixed with ocean hues, never generic museum signage. The badge must stay visually quieter than the Oceanogràfic badge. Placement: lobby selector secondary card and any legacy museum surface. Output: 1024x1024 master, transparent PNG or dark plaque variant, with 512 and 256 exports.

## brand/app-icon-oceanografic

Create the app icon for the Oceanogràfic Jumanji experience. Use a strong rounded-square composition that reads at 48 px: a central eye-of-the-ocean or compass emblem, surrounded by glass, water and gold glow, with a clear marine silhouette and no text. The icon must work for PWA, favicon source and maskable variants. Placement: browser tab, install icon, manifest and sharing previews. Output: 1024x1024 master in PNG, plus exports for 512, 192 and maskable 512.
