# Print

## Global negative prompt

```text
no watermark, no logo, no unreadable text, no clutter, no photo collage, no generic school worksheet
```

## print/passport-cover

Create the front cover of the printable adventure passport for the Oceanogràfic Jumanji route. The cover should feel like a premium field journal from a marine expedition: deep blue-green base, gold trim, central badge space, title zone, name line area and enough calm negative space for printing. It must be portrait A4-friendly and look good both on screen and on paper. Placement: `public/pasaporte.html` cover page. Output: 2480x3508 px at print quality, PNG and WEBP masters.

## print/passport-crystal-sheet

Create a printable crystal sheet that matches the station rewards used in the UI. The sheet should present the crystals in a clean grid with even spacing and enough room for labels or stickers. Style it like a marine expedition sticker board: glassy icons, subtle ocean marks, and large clear slots. Placement: crystal page inside the printable passport. Output: A4 portrait at 2480x3508 px, PNG master.

## print/passport-activity-page

Create a printable activity page for the passport. It should include a route checklist, small boxes or circles for stamping, and calm oceanographic decorative elements around the margins. Keep the center clear for handwriting and stickers. Placement: inside pages of the printed passport. Output: A4 portrait at 2480x3508 px, PNG master.

## print/passport-back-cover

Create the back cover of the printable passport. It should contain a subtle marine expedition frame, a badge placeholder, and a final closing message area with enough breathing room for print. Use the same Oceanogràfic Jumanji visual language and avoid a busy surface. Placement: final page of `public/pasaporte.html`. Output: A4 portrait at 2480x3508 px, PNG master.
