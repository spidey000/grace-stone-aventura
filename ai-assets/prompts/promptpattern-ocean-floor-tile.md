# Pattern: Ocean Floor Tile

## Style Context

**Visual direction:** Oceanogràfic Jumanji — calm ocean floor texture, subtle bubbles, sonar arcs, contour lines, coral dust, soft glass reflections, eelgrass, shell fragments, faint map tracings. Materials: sea stone, aged metal, nautical parchment.

**Palette:**
- Deep background: `#04131D` | Mid background: `#082635`
- Sea surface: `#123B4E` | Elevated surface: `#1B5A6D`
- Ancient gold: `#F1C96B` | Jade: `#56D0B1`
- Deep teal: `#0E8277`

**Rules:** Low contrast, implied motion, perfectly seamless on all edges. No focal point. No text.

## Grok Prompt

**Style:** Seamless vector texture, flat illustration, repeating tile, subtle low-contrast pattern.
**Subject:** A calm ocean floor texture designed to sit behind UI cards without competing. Subtle bubbles drift upward, faint sonar arcs pulse, contour lines trace the seabed. Coral dust particles float, eelgrass sways, shell fragments rest on the sea floor. Faint map tracings like old nautical routes are embedded in the texture.
**Palette:** Deep background `#082635`, muted teal `#123B4E` marks, subtle jade `#56D0B1` highlights at 20% opacity, occasional gold `#F1C96B` micro-accents.
**Mood:** Calm, passive, atmospheric background. Never distracting.
**Composition:** Seamless tile — edges must match perfectly on all four sides. No center focus, no heavy elements. Even distribution across the 1024x1024 square.
**Technical:** Repeatable tile, no visible seams, 1024x1024. Low contrast, soft opacity layers, no sharp lines.

## Negative Prompt

no text, no logo, no focal object, no readable symbols, no clutter, no high-contrast subject, no bright white background, no sharp edges, no modern elements, no photorealistic textures

## Output

- Size: 1024x1024
- Format: WEBP + PNG
- Background: transparent or dark seamless tile
