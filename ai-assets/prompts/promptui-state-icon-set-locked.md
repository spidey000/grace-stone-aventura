# UI: State Icon — Locked

## Style Context

**Visual direction:** Oceanogràfic Jumanji — marine expedition marker. Glass, shell, pearl, sonar language. Readable at 24px. Dark and light surface compatible.

**Palette:**
- Elevated surface: `#1B5A6D` | Deep teal: `#0E8277`
- Text secondary: `#D6CBB9` | Amber: `#D89A2B`
- Ancient gold: `#F1C96B`

**Rules:** No text. Transparent background. Consistent set of 4 states. One icon only.

## Grok Prompt

**Style:** Flat vector icon, minimalist UI glyph, marine expedition marker, sonar language.
**Subject:** A single "locked" state icon for the route step progress. A closed glass or shell container with a subtle lock symbol embedded. The shape is round with a dark teal gradient, muted tones. Sonar ring arcs around the bottom edge at low opacity. No bright colors — this state is inactive.
**Palette:** Dark muted teal `#1B5A6D` base, secondary text `#D6CBB9` for the lock symbol at 50% opacity.
**Mood:** Inactive, waiting, locked. Clear but not attention-seeking.
**Composition:** 96x96 icon, centered. Reads clearly at 24px. Transparent background.
**Technical:** Clean vector shapes, single-color lock symbol, low contrast. Part of a 4-icon set (locked, active, completed, done).

## Negative Prompt

no text, no readable letters, no watermark, no logo, no clutter, no photorealistic, no flat generic UI circle, no bright colors, no gradients that hide the shape

## Output

- Master: 96x96
- Additional: 48x48, 32x32
- Format: PNG with transparent background
