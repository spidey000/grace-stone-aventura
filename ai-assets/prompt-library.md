# Prompt Library

Prompts base para generar los assets con IA. Sustituir las variables entre llaves y mantener el negative prompt global.

## Global Negative Prompt

```text
no readable text, no random letters, no watermark, no logo, no brand marks, no gore, no realistic weapons, no horror, no photorealistic child, no cluttered composition, no low contrast, no blurry details, no generic flat vector, no neon sci-fi, no modern city, no beige-only palette
```

## Backgrounds

```text
Create a premium adventure game background for {asset_name}, inspired by a mysterious jungle board-game expedition, ancient ruins, hidden treasure, warm gold highlights and jade magical accents. Composition: {composition}. Mood: {mood}. Use a cinematic illustrated style, rich but controlled texture, readable dark areas for UI, detailed edges, safe empty center, no characters, no text. Color palette: deep jungle blue-green, aged gold, jade glow, warm parchment accents. Output: {size}, {output_format}.
```

## Patterns

```text
Create a seamless tileable pattern for {asset_name}, inspired by carved jungle symbols, aged map marks, vines, compass lines and subtle treasure motifs. Low contrast, premium illustrated texture, suitable behind UI text, no focal object, no text, no logo. Palette based on dark teal, muted jade, aged gold and soft parchment. Output: square tile, {size}, {output_format}.
```

## Step Icons

```text
Create a crisp adventure UI icon for {asset_name} in state {state}. The icon must be legible at 24 px, with a simple bold silhouette, carved board-game style, subtle gold/jade lighting, transparent background, no text. Use shape language to communicate the state, not only color. Output: centered icon, {size}, {output_format}.
```

## Key Object Icons

```text
Create a single collectible adventure object icon: {asset_name}. Centered object, transparent background, premium illustrated style, subtle worn edges, soft shadow, gold and jade accents, readable silhouette at small sizes, no text. The object should feel like it belongs in a jungle treasure board-game world. Output: {size}, {output_format}.
```

## Avatars

```text
Create a stylized family-friendly adventure avatar of {asset_name} with expression {state}. Bust portrait, consistent illustrated game style, warm cinematic lighting, jungle expedition mood, readable face, no text, no logo, not photorealistic. Use deep teal shadows, gold rim light and subtle jade magic accents. Output: {size}, {output_format}.
```

## Animation Keyframes

```text

Provide distinct frames for start, anticipation, reveal, impact and settle. Keep object position stable enough for sprite animation. Output: {size}, {output_format}.
```
